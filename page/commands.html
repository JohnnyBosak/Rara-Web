Command List

  • /advanced-help: Obtain detailed assistance.
  • /automod: Set up the Automod System.
  • /banner: Create a stylish banner for a user.
  • /database: Database settings.
  • /guild list: Display a list of all servers Rara is in.
  • /dm: Send a direct message.
  • /emoji mixer: Combine two different emojis.
  • /giveaway: Initiate or configure existing giveaways.
  • /chatgpt: Ask Chat-GPT a question.
  • /help: List all commands
  • /memberinfo: View your or any member profile's information.
  • /ping: Check the bot's latency.
  • /play: Play some music
  • /poll: Create a poll.
  • /clear: Delete multiple messages
  • /reload: Reload your commands/events.
  • /say: Make the bot say something.
  • /setup ban_log: Set up the logging channel for the ban/unban event.
  • /setup booster: Set up server boost logging system.
  • /setup clear_log: Set up logging for the /clear command.
  • /setup message_log: Set up the messages logging system for your guild.
  • /setup welcome: Set up the welcome member logging system for your guild.
  • /stealsticker: Get the message's sticker.
  • /timeout: Temporarily restrict a member's ability to communicate.
  • /vote-kick: Initiate a vote to kick a member from a voice channel.
  • /yosoundboard: Play a sound from the soundboard.
